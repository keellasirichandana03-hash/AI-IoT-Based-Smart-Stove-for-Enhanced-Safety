# AI-IoT-Based-Smart-Stove-for-Enhanced-Safety
Developed an AI-IoT Smart Stove Safety System using CNN-based real-time age detection. If age &lt;7, servo motor auto turns off stove knob. Added gas sensor at LPG cylinder; on gas detection, second servo motor shuts cylinder valve. Built using Python, Deep Learning, OpenCV, Arduino &amp; IoT.
